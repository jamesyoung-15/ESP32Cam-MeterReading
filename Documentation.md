## ESP32-Camera Setup



The ESP32-Camera prepares a JSON the following format:
```JSON
{
    "S3Folder": "Folder",
    "cropLeft": 0, 
    "cropTop": 0,
    "cropHeight": 0,
    "cropWidth": 0,
    "base64Image": ,
}
```
